## 3. 카드 불변 규칙 (지금부터 유효)

1. **카드 방향: 세로 고정 (v0.3)** — 기본 비율 **5:7** (실물 트레이딩 카드 표준), 허용 범위 0.68~0.75. 가로 카드 금지. 카드의 멘탈 모델 자체가 세로이고, 인물 사진 크롭·모바일 2열 그리드·OG 5장 가로 배열 모두 세로가 유리.
2. **OVR이 영웅**: 카드 폭의 35~40% 크기, 자간 타이트(-0.02em 내외), 디스플레이 폰트.
3. **단색 평면 금지**: 모든 카드 배경은 미세 그라디언트 + 그레인 텍스처 (SVG noise, opacity 0.04~0.08).
4. **사진 처리 레시피 v3 (필수)** — 기본은 **원본 컬러** (유니폼 색 = 팀 정체성 정보, v0.4 결정). 편차 흡수는 채도가 아니라 페이드·크롭·프레임이 담당:
   ```css
   /* 기본안: 풀컬러 + 미세 정규화 */
   img { filter: contrast(1.03) brightness(.97); }
   /* 하단 페이드 (필수 — 통일감의 핵심 장치) */
   linear-gradient(180deg, transparent 40%, rgba(bg-base, .65) 95%)
   /* 카드 톤 통일용 얇은 오버레이 */
   rgba(bg-base, .10)
   ```
   lab A/B 3안: ① 풀컬러 + 페이드(기본안) ② 저채도 컬러(saturate .75 — 시대 편차가 거슬릴 때의 후퇴안) ③ 흑백 + 골드 틴트 duotone — 실사진 20장으로 수렴 후 1안 고정. 페이드 없는 무보정 직접 노출은 금지.
   **누끼(컷아웃) 트랙 (v0.5)**: 배경 제거가 샘플 테스트(CURSOR_GUIDE §5-2)를 통과하면 **누끼가 기본** — 배경 편차를 가리는 게 아니라 제거하는 가장 강한 통일 장치. 컴포지션: 사진 박스 없이 카드 배경 위 직접 배치, 하단 페이드로 카드에 블렌드, 머리 상단이 OVR 영역을 살짝 침범 가능(깊이감), 왕관은 머리에 직접 얹힘. 채택 단위는 전체/연도 게이트만 — 카드별 누끼·사각 혼재 금지.
5. **일반 프레임 95% 룰**: 화면 카드의 95%+가 일반 프레임. 일반 카드의 완성도 > 특수 효과. 특수 레이어 작업은 일반 카드 확정 후에만.
6. 보더: 1px 단색 대신 저대비 그라디언트 또는 이중 보더 (골드 계열 보더는 WORLDS 프레임 전용).
7. 그림자: 카드 외곽 1개만 (큰 blur + 낮은 opacity). 내부 요소 그림자 금지.
8. 사진 없음 폴백: 이니셜 1자 + 팀 컬러 그라디언트 원형 — 사진 카드와 같은 채도 레벨 유지.
9. **골드 예산 (v0.2)**: 골드는 **업적 신호 전용** — WORLDS 프레임, 배지·라벨 아이콘, 결과 화면 트로피에만 허용. 일반 카드의 보더·이름·OVR에 골드 사용 금지. 전부 골드면 아무것도 특별하지 않다 — 한 화면 기준 골드 체감 면적 5% 이하 유지.

## 4. 장식 레이어 비주얼 스펙

### 4.1 WORLDS 프레임 (사이트 유일의 반짝이)
- 그라디언트 보더(2px) + 시머 스윕: 주기 2.8s = 스윕 1.2s + 휴지 1.6s, 스윕 opacity 0.12~0.18. 쉴 새 없이 번쩍이면 싸 보임 — 휴지가 핵심.

### 4.2 왕관 (밈 레이어)
- **에셋 제작 절차 (고정)**: 호빈이 그림판/메모앱에서 30초 손그림 → 투명 PNG (256px 내외) → `/public/crown.png` 동결. AI 재생성·보정·변형 **금지**.
  - 대안(손그림 불가 시): AI에 결함 주입 프롬프트로 5안 생성("좌우 비대칭, 꼭짓점 홀수 개, 선 굵기 불균일, 채색 테두리 밖으로") → **가장 못 그린 안** 채택.
- 배치: 사진 좌상단, `rotate(-12deg)`, 폭 = 사진 폭의 42~48%, 사진 영역에서 살짝 삐져나오게.
- 전 카드 동일 에셋·동일 배치 — "일관된 조악함"이 룰.

### 4.3 MSI 라벨
- 사진 하단 소형 캡슐, `MSI WINNER` 영문 고정, 10~11px, accent-gold 저채도 버전. 왕관과 시각 무게 경쟁 금지 (라벨은 조용하게).

### 4.4 배지 (리그 우승 / All-Pro 1st)
- 우상단 16px 아이콘 최대 2개. 자체 SVG 아이콘 (외부 아이콘 팩 금지).
