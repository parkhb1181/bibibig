# GAME_SPEC_v1 — 시뮬·밸런스·결과화면 확정 사양

> 2026-06-12 호빈 확정. PRD §시뮬 규칙의 보강 SSOT.
> 데이터 파이프라인(02/03 진행 중)과 독립 — 프론트는 더미 데이터로 선행 구현, 실데이터 완성 후 연결.

---

## 1. 온보딩 — 마찰 제로

- **접속 즉시 자동 스핀**: 랜딩에서 설명·버튼 없이 5인 스핀이 바로 돌아간다.
- 첫 판이 30초 안에 스핀 → 드래프트 → 시즌 시작까지 흐르게 한다 (설명 불필요, 해보면 이해).

## 2. 리롤 — 팀 전체 1회

- 5인 팀이 떴을 때 **팀 전체를 통째로 1회** 다시 뽑을 수 있다 (주사위 버튼 1개).
- 카드 개별 리롤 아님. 선택 단계 없이 버튼 1탭으로 즉시 재스핀.
- 리롤 후에는 그 팀으로 확정 (마음에 안 들어도 진행).
- v1 출시 후 피드백으로 횟수 튜닝 (1회 → 조정 가능 구조로 구현).

## 3. 시즌 구조·진출 규칙

```
스프링 (국내) → MSI (국제) → 서머 (국내) → Worlds (국제)
```

| 단계 | 진출 조건 | 결과 구분 |
|---|---|---|
| 스프링 | 무조건 참가 | 우승 / 준우승 / 4강 / 탈락 |
| MSI | **스프링 2위 이내** | 우승 / 패배(상대팀 표시) / 미진출 |
| 서머 | 무조건 참가 | 우승 / 준우승 / 4강 / 탈락 |
| Worlds | **서머 4위 이내** | 우승 / 패배(상대팀 표시) / 미진출 |

- 미진출 단계는 **회색** 처리 ("진출 실패 (스프링 3위)" 등 사유 표기). 패배(빨강)와 구분.

## 4. 승패 판정 — 확률 기반

- 본인 팀 5인 OVR 합산 → 팀 전력치 → 라운드 상대(opponents-2026.json 레이팅)와 비교.
- OVR 차이를 **승률로 변환하는 완만한 곡선**. 확률 판정 (몬테카를로 09 보정값 활용).
- 라운드별 상대는 opponents 풀에서 매칭. 국제 상대가 국내보다 강하게 (intl 풀: CLG-16 86 ~ SKT-16 97).

### 곡선 기준점 (확정)

- **약팀(OVR 80) vs 강팀(OVR 92) 단판 → 약팀 승률 20%** ← 언더독 반란 기준선
- 이 점을 지나도록 OVR차→승률 곡선을 그린다. 차이가 크면 더 낮게(80 vs 99 ≈ 5~8%), 작으면 높게(85 vs 88 ≈ 40%대).

## 5. OVR 분포 가이드 (카드 풀)

| OVR | 의미 | 국제 무대 기대치 |
|---|---|---|
| 90대 | 우승후보 | Worlds/MSI 우승권 |
| 80 중반 | 강팀 | 8강~4강 |
| 80 초반 | 평범 — "똥팀"의 정체 | **진출은 하되 조별 광탈** |
| 70대 이하 | 약체 | 국내 광탈, 국제 못 감 |

- 똥팀 = "무명 약체"가 아니라 **아는 선수의 약한 시즌 / 조별 광탈급 팀**. 듣보 카드로 채우지 않는다 (카드 풀 필터: LCS/LEC 국내 v1 제외, 서브 제외 — 별도 확정).
- 99는 극소수 (앵커: Faker 16, Canyon 20, Chovy 24 등), 75~85 구간이 두텁게.

## 6. 등급 체계

```
GRAND SLAM   👑  스프링+MSI+서머+Worlds 4관왕 (유저 최종 목표)
WORLDS CHAMPION  Worlds 우승 (4관왕 아님)
MSI CHAMPION     MSI 우승, Worlds 실패
DOMESTIC KING    국내 우승, 국제 무관
PLAYOFF          플옵권, 무관
REBUILD          전부 광탈
```

- PRD에 기존 등급 정의가 있으면 그것이 우선. 없으면 위 골격 사용.
- GRAND SLAM 달성 시 왕관 오버레이 (crown.png — 흑백 낙서풍, 전 카드 동일 위치 고정, 어두운 배경엔 흰 테두리).

## 7. 결과 화면 — 스크린샷 공유 최적화 (신규 구현)

한 화면(한 프레임 캡처)에 다음이 모두 보이게:

1. **5인 카드** — 시즌별 사진 + OVR
2. **4단계 결과 타임라인**
   - 승 = 🟢 초록 / 패 = 🔴 빨강 / 미진출 = ⚪ 회색
   - 패배: `vs {팀명} ({연도}) — 패` (opponents-2026.json 라벨 재활용)
   - 우승: `vs {결승상대} 격파`
3. **최종 등급** + 왕관(우승급)

예시:
```
스프링   🟢 우승
MSI      🔴 패 — vs RNG (89)
서머     🟢 우승
Worlds   🟢 우승 — vs SKT-16 (97) 격파

최종 등급: WORLDS CHAMPION
```

## 8. 밸런스 검증 지표 — 데이터 완성 후 1만 판 시뮬

아래 3개가 **동시에** 만족되게 곡선 튜닝. 안 되면 20% 기준점부터 조정.

| # | 지표 | 목표 |
|---|---|---|
| 1 | 그랜드슬램(4관왕) 전체 빈도 | **100판당 1~2회** (랜덤 스핀+리롤 1회 포함 기준) |
| 2 | 언더독 반란 | 80 약팀이 92 강팀 단판 ~20% 승리. 약팀 월즈 4강 도달이 가끔 나옴 |
| 3 | 강팀 정당성 | 90대 우승후보의 Worlds 우승률 20~30% — 강팀도 삐끗하되 "맨날 광탈" 아님 |

- 주의: 리롤 1회가 분포를 위로 끌어올리므로, 시뮬은 **리롤 포함 행동 모델**(약팀이면 리롤한다고 가정)로 측정.
- 20% 기준은 15%보다 이변이 잦으므로 지표 1(그랜드슬램 빈도)이 흔들릴 수 있음 — 시뮬에서 충돌 시 국제 라운드 난이도(intl 상대 레이팅)로 보정.

## 9. 출시 후 튜닝 항목 (v1에 박고 측정)

- 리롤 횟수 (1회 → 피드백)
- 승률 곡선 기울기 / 20% 기준점
- 그랜드슬램 빈도 실측 대비 조정
- 잔여 awards `# ?` 보강 (비앵커, +4~6 영향)
